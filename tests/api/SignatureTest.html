JavaMail API Signature Test

This is the API signature test for the entire jakarta.mail.* package. This test verifies the existence of all expected public classes and interfaces. In those public classes and interfaces it verifies the existence, correct modifiers, arguments, and throws clauses of public constructors and methods. It also verifies the existence and correct modifiers of public fields.

This test is version specific. The current version is 2.1.
