JavaMail API Signature Test

This is the API signature test for the entire javax.mail.* package. This test verifies the existence of all expected public classes and interfaces. In those public classes and interfaces it verifies the existence, correct modifiers, arguments, and throws clauses of public constructors and methods. It also verifies the existence and correct modifiers of public fields.

This test is version specific. The current version is 1.6.

title API Signature verification
name Mail_SigTest
executeClass com.sun.tdk.signaturetest.SignatureTest
executeArgs -ClassPath $LOCAL_CLASSES -TestURL $testURL -Package javax.mail -FileName javax.mail_1.6_java_${java.specification.version}.sig
