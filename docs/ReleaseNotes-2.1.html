Jakarta Mail Technology Compatibility Kit, Version 2.1

Release Notes, December, 2021

Kit Contents

The Jakarta Mail, Version 2.1 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta Mail TCK tests signature, API, and pluggability tests:

    • A signature test that checks that all of the public APIs are supported in the Jakarta Mail Version 2.1 implementation that is being tested
    • API tests to verify the functionality of all the public APIs under
      • jakarta.mail
      • jakarta.mail.event
      • jakarta.mail.internet
      • jakarta.mail.search
      • jakarta.mail.util
    • Pluggability tests for testing the SPI provider interface for supplying your own MailcapRegistryProvider and MimeTypeRegistryProvider

Jakarta Mail TCK Facts

The test suite bundle contains the following:

  • 1 signature test
  • 321 API tests
  • 2 pluggability tests

Platform Notes

The Jakarta Mail TCK tests have been built and tested with JavaTM SE 11.

The Jakarta Mail TCK tests have been run on the following platforms:

  • Centos Linux 7
  • Alpine Linux v3.12

The Jakarta Mail TCK tests have been run against a Jakarta mail 2.1 Compatible Implementation, Eclipse Angus 1.0.0 and Eclipse GlassFish 6.2.3 or 7.0.0
