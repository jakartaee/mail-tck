Guide to Jakarta Mail TCK 2.1 Documentation

The Jakarta Mail TCK 2.1 documentation includes the following:

  • The Jakarta Mail TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
    • Updates to the Release Notes, if needed, are available from the Jakarta Mail Specification Web Site: https://jakarta.ee/specifications/mail/2.1/
  • The Jakarta Mail Technology Compatibility Kit, Version 2.1 User's Guide (or in PDF) provides the information that you need to install, set up, and run the Jakarta Mail TCK, Version 2.1. In addition, the guide provides the rules you must comply with to pass the Jakarta Mail TCK.
